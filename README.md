# sim_rbtree

How to execute the simulator?

 - orig version: 1. make </br>
                 2. ./execute WSS [|rand]</br>
                 ex 1.) sequential version: ./execute 10000000</br>
                    2.) random version: ./execute 10000000 rand</br>
 
 - bigarr version: 1. make</br>
                   2. ./execute GroupSize WSS [|rand]</br>
                   ex 1.) sequential version: ./execute 10 10000000</br>
                   ex 2.) random version: ./execute 10 10000000 rand</br>
