# sim_rbtree

How to execute the simulator?

 - orig version: 1. make
                 2. ./execute WSS [|rand]
                 ex 1.) sequential version: ./execute 10000000
                    2.) random version: ./execute 10000000 rand
 
 - bigarr version: 1. make
                   2. ./execute GroupSize WSS [|rand]
                   ex 1.) sequential version: ./execute 10 10000000
                   ex 2.) random version: ./execute 10 10000000 rand
